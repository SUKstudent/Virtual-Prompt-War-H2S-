### 🎤 IntelliCrowd AI – Smart Event Coordination Platform

An AI-inspired crowd management system designed to improve crowd movement, reduce waiting times, and enable real-time coordination at large-scale events like concerts.

----

## 🚀 Project Overview

This project presents a smart system that helps manage crowd flow in large events. It uses real-time data insights to guide attendees, minimize congestion, and improve overall experience. The prototype demonstrates how intelligent recommendations can enhance safety and efficiency in crowded environments.

----

## 📊 Input Features

The system considers the following inputs (simulated in prototype):

👤 User Location (inside venue)

📍 Area Zones (Stage, Food Court, Entry Gates, etc.)

👥 Crowd Density Data

⏱ Queue Status (waiting time at stalls/gates)

----

## 🎯 Output

📍 Suggested less crowded routes

⏱ Estimated waiting times

🚨 Alerts for overcrowded areas

📊 Real-time crowd insights

🖥️ Live Prototype

The application is deployed using Google Cloud Run.
